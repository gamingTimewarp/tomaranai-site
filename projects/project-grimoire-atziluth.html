Grimoire Atziluth

March 2026 · v1.0.0

An offline-first esoteric reference and divination app for Linux, Windows, and Android. It combines an interactive reading journal, a comprehensive occult knowledge base, live astrological calculations, and spaced-repetition study tools — all running entirely on-device with no cloud dependency, no account required, and no data leaving your device.

The name comes from Atziluth — the highest of the Four Worlds in Qabalah, the world of pure emanation.

↓ Download (Linux · Windows · Android) ⌥ Source on GitHub

Linux: .AppImage (portable) or .deb / .rpm  ·  Windows: .exe installer or .msi  ·  Android: universal .apk (sideload)


Divination & Readings

  • 11 built-in decks — Rider-Waite-Smith, Thoth, Tarot de Marseille, Etteilla, Elder Futhark Runes, Lenormand, Ogham, Geomancy, Mahjong Oracle, Playing Cards, Tea Leaf Symbols
  • 14 spreads including Celtic Cross, Tree of Life, Grand Tableau, Zodiac Year, and free-form drawing
  • Reversals, clarifier cards, per-position keyword panel
  • Intention/question field stored with each reading
  • Astrological sky snapshot captured automatically at the moment of every save
  • Export any reading as Markdown or a PNG image

Journal

  • Combined reading + journal timeline, reverse-chronological with filter search
  • Rich text editor — headings, bold, italic, lists, blockquotes
  • Entity links — tag any reference entity to a reading or entry
  • Soft-delete with 5-second undo; spread visualisations inline in the expanded view

Reference

  • 24+ entity types, 35+ browse categories, real-time full-text search
  • Multi-tradition attributions — toggle which esoteric frameworks are active
  • Personal annotations stored locally on each entity page
  • Bookmarks, recently viewed strip, random entity navigation

Astrology

  • Natal charts with SVG wheel, planet/house table, aspect table, mutual receptions, Arabic parts
  • Current sky panel with 5-minute updates and transit-to-natal aspects
  • Astronomical calendar — ingresses, retrograde stations, eclipses, Moon phases
  • Tropical, Sidereal, and IAU 13-sign modes; 6 house systems; Chiron + major asteroids
  • All calculations offline via astronomy-engine — no internet required

Qabalah

  • Interactive SVG Tree of Life (Queen Scale colours) and Nightside Tree (Qliphoth)
  • Gematria calculator with Hebrew letter input and real-time total
  • Numerology calculator — Expression, Soul Urge, Personality, Life Path — Pythagorean and Chaldean

Study

  • SM-2 spaced repetition across any entity type
  • 4 question modes: flashcard, multiple choice, fill-in-blank, image recognition
  • 14-day accuracy sparkline, streak tracking, per-type progress breakdown

Customisation

  • 7 theme presets + full colour token editor (18 tokens, live preview)
  • 6 accessibility modes — colour vision, dyslexia font, reduced motion, card captions
  • Custom entities, spreads, decks, and traditions
  • Custom CSS injection
